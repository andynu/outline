How to compare

  1. Open each benchmark in a new tab
  2. Hard refresh (Ctrl+Shift+R) to clear cache
  3. Note the "Total render" time in the stats box (top-right)
  4. Refresh several times to get consistent readings

Note: These benchmarks measure initial render only. For update performance, consider the actual editing experience in the app.
